# Sorting Visualization

![Image of webiste](https://i.ibb.co/m6yPcDT/Screenshot-3.png)

## How to install

```bash
  git clone https://github.com/sk0le/sorting-visualization.git

  cd sorting-visualization

  npm install

  npm start
```
